# AI-Driven Construction Cost Estimation

This project is a full-stack application with a FastAPI backend and React frontend for estimating construction costs with upgrade options.

## Backend (FastAPI)
- Located in the `backend` folder
- Main entry: `main.py`
- Install dependencies: `pip install -r requirements.txt`
- Run server: `uvicorn main:app --host 0.0.0.0 --port 8000`

## Frontend (React)
- Located in the `frontend` folder
- Main entry: `App.jsx`
- Install dependencies: `npm install`
- Run frontend: `npm run dev` (uses `serve` for static files)

## Usage
1. Start the backend server
2. Start the frontend
3. Enter dimensions, bedrooms, and upgrades to get a cost estimate

For any issues, please check the backend and frontend logs for errors.
